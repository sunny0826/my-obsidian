## 问题描述

- **任务名称**: 整理记忆
- **Cron**: 0 20 * * 0 (每周日 20:00)
- **最后运行**: 2026-02-22 20:00:00
- **状态**: error
- **执行时长**: 124746ms (124s)
- **原因**: 任务执行时间接近 120s 超时限制

## 解决方案

需要删除原任务并重新创建，增加超时时间到 300000ms (5分钟)：

```bash
# 删除原任务
openclaw cron delete 0974151c-2199-4660-8f87-2c9165bf4aaf

# 重新创建，增加超时时间
openclaw cron create --name "整理记忆" --cron "0 20 * * 0" --timeout 300000 --type isolated
```
